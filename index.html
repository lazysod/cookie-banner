Cookie Consent v0.1.0

Framework-agnostic, 4.5kb, actually blocks scripts. MIT licensed.

Test It

1. Open DevTools Console

2. You should see no logs yet

3. Click "Accept all" below

4. Console will show: "Analytics fired" + "Marketing fired"

Install

<script src="cookie-consent.js"
        data-config='{"lang":"auto","localesPath":"./locales/"}'></script>

  <!-- Wrap scripts you want to block -->
  <script type="text/plain" data-category="analytics" data-src="gtag.js"></script>

JS API

CookieConsent.hasConsent('analytics') // true/false
CookieConsent.show() // reopen banner
window.addEventListener('cc:consent', e => console.log(e.detail))
